1. 기본 속성 선택자

    기본 선택자 뒤에 []를 사용하여 속성과 속성값을 사용한다.

    선택자[속성=속성값] {
      스타일 속성: 값;
      스타일 속성: 값;
      ...
    }

3. 동위(형제) 선택자

    동위 관계(같은 레벨)에서 뒤에 위치한 태그를 선택할 때 사용한다.

    1) a 선택자 뒤에 있는 b 요소 한 개 선택
          a+b {
              스타일 속성: 값;
              스타일 속성: 값;
              ...
          }

    2) a 선택자 뒤에 있는 모든 b 요소 선택
          a~b {
              스타일 속성: 값;
              스타일 속성: 값;
              ...
          }

4. 반응 선택자

    사용자의 움직임에 따라 달라지는 선택자

    1) 사용자가 클릭하는 요소에 스타일 적용
        선택자:active {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }

    2) 사용자가 마우스를 올려놓는 요소에 스타일 적용
        선택자:hover {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }

5. 상태 선택자

    입력 양식의 상태에 따라 선택되는 선택자    

    1) 체크된 상태의 태그
        선택자:checked {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }

    2) 초점이 맞춰진 태그 스타일 변경
        선택자:focus {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }

    3) 사용 가능한 태그 선택
        선택자:enabled {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }

    4) 사용 불가능한 태그 선택
        선택자:disbled {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }
