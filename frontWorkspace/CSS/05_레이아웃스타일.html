포지셔닝

    position

    static : 기본값
              HTML 작성 순서대로 쌓임 (블록이 쌓이듯 ...)

    relative : 기존 자기자신 위치를 기준으로 위치선정
               (그렇다고, 다른애가 내 원래자리 침범하는건 안됨)

    absolute : 부모태그를 기준으로 위치선정
              (부모의 조건 : position 값이 static 이 아니어야 함)
              (부모조건에 맞지 않으면, 상위요소를 찾아감...반복)
              (찾다가 찾다가 없으면,,, 결국 html 요소까지 올라감)
              유령이 됨 (==다른 요소와 충돌 없음 , 기존자리 차지안함)

    fixed : 화면 기준 위치선정
            유령이 됨 (==다른 요소와 충돌 없음 , 기존자리 차지안함)

    sticky : 조건부 fixed
              기본적으로는 자기 자리 지킴
              근데, 화면 기준으로 자리잡을 수 있으면 fixed 가 됨
              부모요소를 벗어날 수 없음
